# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Commands

```bash
bun dev          # start dev server
bun build        # production build
bun lint         # ESLint
bun format       # Prettier (writes in place)
```

No test suite is configured.

## Architecture

**Stack:** TanStack Start (SSR) + TanStack Router (file-based) + React 19 + Tailwind CSS v4 + shadcn/ui (New York style).

### Routing

All routes live in `src/routes/`. The file name is the URL segment — no `/pages/` or Next.js conventions. `routeTree.gen.ts` is auto-generated by the router plugin; never edit it manually. See `src/routes/README.md` for the full naming convention table.

- `src/routes/__root.tsx` — app shell (QueryClientProvider, global head tags, error/404 pages). The `<Outlet />` inside must not be removed.
- `src/router.tsx` — creates the router with a shared `QueryClient` passed through route context.

Each page route wraps its content in `<SiteLayout>` from `src/components/SiteLayout.tsx`, which provides the sticky navbar, footer, and an optional `<PageHero>` export for inner-page headers.

### Styling

Tailwind CSS v4 with a custom design token layer defined in `src/styles.css`. The brand palette is exposed as Tailwind utilities:

| Token | Usage |
|---|---|
| `clinical-blue` | Primary dark violet — headings, navbar, footer, CTA buttons |
| `clinical-accent` | Bright violet — highlights, eyebrows, active states |
| `clinical-slate` | Muted body text |

shadcn/ui components are in `src/components/ui/` and use the same CSS variables. Add new shadcn components with `bunx shadcn add <component>`.

### Server / Client boundary

Files named `*.server.ts` are server-only (never bundled into the client). Public env vars must use the `VITE_` prefix; secret env vars go in `*.server.ts` files and must be read **inside a function**, not at module scope (required for Cloudflare Workers compatibility).

### Path aliases

`@/` resolves to `src/` (configured via `vite-tsconfig-paths`).
